== AwkwardTurtle

=== Description

Awkward turtle awkwardly animates a turtle on your terminal.

It might be incompatible with your terminal, which is, well, awkward!

=== Installation

  gem install awkward_turtle

=== Usage

At a command prompt:

  awkward_turtle
